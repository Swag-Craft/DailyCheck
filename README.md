# 日常 DailyCheck


## 日常 DailyCheck 是什么

日常 DailyCheck  一个轻量级可爱的打卡小工具。

## 功能特性

**今天任务打卡** 

**每天习惯打卡** 

**查看过往打卡情况** 

**多种主题配色** 

**横版和竖版打卡卡片**
