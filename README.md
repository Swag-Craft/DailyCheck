# 日常 DailyCheck


## 日常 DailyCheck 是什麼？

一個可愛的桌面打卡小工具，幫你記錄任務、習慣、每日日記和自定義頁面資訊。


## 功能特性

# 📋 今日任務 — 添加、完成、刪除，進度一目了然
# 🌸 習慣打卡 — 自訂 emoji 圖標 + 每日目標次數，×/✓ 按鈕即時計數，進度條視覺化
# 📝 每日日記 — 固定在頁面下方，隨手寫下當天心情
# 📅 過往記錄 — 按月回看所有任務與習慣，編輯模式可刪改舊記錄
# 👁️ 頁面監控 — 貼上任意網址，全屏框選關注區域，即時卡片顯示，拖動位置自動儲存
# 🐱桌面精靈 — sprite 動畫、拖動位置記憶、氣泡對話、單擊/雙擊互動、右鍵菜單、跨天自動隱藏
# 🎨 11 套主題 — 草香、大地手帳、暖紙、櫻花粉、素白、青夜、深夜藍、深海、沉思、賽博朋克、極光炸裂
# 🌐 六語支援 — 簡體中文 / 繁體中文 / English / Français / Deutsch / Русский
