# 日常 DailyCheck


## 日常 DailyCheck 是什麼？

一個可愛的桌面打卡小工具，幫你記錄任務、習慣、每日日記和自定義頁面資訊。


## 功能特性

### 📋 今日任務 
### 🌸 習慣打卡 
### 📝 每日日記 
### 📅 過往記錄 
### 👁️ 頁面監控 
### 🐱桌面精靈 
### 🎨 11 套主題
